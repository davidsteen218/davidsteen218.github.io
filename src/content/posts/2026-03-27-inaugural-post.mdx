## GitHub Pages

The backbone of this site specifically, is GitHub Pages, GitHub's webpage hosting service.  This is wholely inspired by (NetworkChuck's 2024 Blog Launch)[https://youtu.be/dnE7c0ELEH8] and a hope that by sharing what I learn, I can help others while codifying the information deeper in my mind.  

## CodeSpaces

Today's epiphany leverages yesterday's discovery.  GitHub allows one user one free codespace.  This provides a Visual Studio style, web based IDE (Integrated Development Environment) accessible anywhere you have internet.  This significantly streamlines the previous need to cascade dev environments between Windows 11, WSL, and homelab devices running Linux.  

## Markdown

Having grown up around tech, the need for filfe readability across platforms, operating systems, and updates makes Markdown the best choice for the current homelab renovations (Laying a good foundation for the work ahead).  Linux config files are text files.  So are Markdown files.  Layering in the readability for LLM's makes Markdown the best layout for an electronic lab notebook for future homelab dev discovery.  

On a personal note, in high school I realized I liked the food I grew up eating and that would not continue when I left for college.  This kicked off a project to type up my Mom's recipes so that I could take them with me.  In subsequent years, the original files were formatted to .txt with a natural next step being Markdown.
